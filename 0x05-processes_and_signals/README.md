#### Technologies

* Bash:     5.1-6ubuntu1
* Docker:   24.0.5
* GCC:      9.4.0
* Ubuntu:   20.04 LTS

#### Learning Objectives

* At the end of this project, you are expected to be able to `explain to anyone`, **without the help of Google**:
    * What is a PID
    * What is a process
    * How to find a process' PID
    * How to kill a process
    * What is a signal
    * What are the 2 signals that cannot be ignored
